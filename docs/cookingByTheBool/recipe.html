How to Make an HTML Recipe

  1. Make a new HTML file Sublime

  2. Type in "<" "HTML" then press enter when the auto fill menu comes down

  3. In between the title tags, enter the title of the document which will appear on the tab

  4. In the body, make a header outlining what the recipe is for

  5. Still in the body, but not in the header, type "<" "ol" >" for an ordered list, or type in "<" "ul" >" for an unordered list

  6. Inside the tags for the list you have created, enter as many list items necessary for the process you are describing

  7. Apply any effects to the text you wish to make

  8. Make sure all tags are closed

  9. Make sure to save
